Adafruit QSPI Library
Adafruit QSPI library

Introduction

This is documentation for Adafruit's library for interfacing with the QSPI peripheral and external flashes on SAMD51 and nRF52 microcontrollers. This is intended to be used with the Adafruit Bluefruit nRF52840, Adafruit Metro M4 Express, Adafruit Feather M4 Express, and other M4 Express development boards.

Adafruit invests time and resources providing this open source code, please support Adafruit and open-source hardware by purchasing products from Adafruit!
